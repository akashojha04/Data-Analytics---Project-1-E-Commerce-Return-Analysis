# E-Commerce Return Prediction & Analytics Dashboard

## Project Overview

This project focuses on analyzing e-commerce order data to understand product return behavior and build a machine learning model to predict return probability. It combines **data analysis, machine learning, and business intelligence** to provide actionable insights that help reduce return rates and improve decision-making.

---

## Objective

* Analyze patterns behind product returns
* Identify key factors influencing returns
* Build a predictive model to classify return-prone orders
* Develop an interactive dashboard for business insights

---

## Tech Stack

* **Python** – Data processing, EDA, and machine learning
* **SQL (SQLite)** – Return analysis queries
* **Power BI** – Interactive dashboard visualization
* **Jupyter Notebook** – Development environment

---

## Workflow

### 1. Data Cleaning & Preprocessing

* Handled missing values and incorrect data types
* Prepared dataset for analysis and modeling

---

### 2. SQL-Based Return Analysis

* Calculated return rates and trends
* Analyzed returns by category, location, and pricing

---

### 3. Exploratory Data Analysis (EDA)

* Visualized patterns using plots
* Identified relationships between returns and features

---

### 4. Feature Engineering

* Created time-based and behavioral features
* Encoded categorical variables

---

### 5. Model Building

* Built **Logistic Regression** and **Random Forest** models
* Compared performance across models

---

### 6. Model Tuning & Evaluation

* Tuned Random Forest for better performance
* Evaluated using:

  * Accuracy
  * Precision
  * Recall
  * F1 Score

---

### 7. Prediction & Risk Classification

* Generated return probabilities
* Classified orders into:

  * 🔴 High Risk (>70%)
  * 🟡 Medium Risk (40–70%)
  * 🟢 Low Risk (<40%)

---

### 8. Dashboard & Business Insights

* Built a **4-page Power BI dashboard**:

  * Overview
  * Return Analysis
  * Customer Insights
  * ML Insights
* Derived actionable recommendations to reduce returns

---

## Key Insights

* High discounts significantly increase return rates
* Certain product categories contribute most to returns
* Cash on Delivery (COD) orders show higher return probability
* Higher-priced products tend to have higher return risk

---

## Machine Learning Model

* **Model Used:** Random Forest Classifier
* **Accuracy:** ~65%
* **Focus Metric:** Recall (to capture more return-prone orders)

---

## Dashboard Features

* KPI metrics (Total Orders, Returns, Return Rate)
* Category, city, and payment-based analysis
* Customer behavior insights (age, gender, order patterns)
* Risk-level segmentation using ML predictions

---

## Business Recommendations

* Limit excessive discounts on high-return products
* Promote prepaid orders over COD
* Improve product descriptions to reduce mismatch
* Monitor high-risk orders using prediction model

---

## Project Outcome

This project demonstrates an **end-to-end data science workflow**, combining analytics and machine learning to solve a real-world business problem and provide actionable insights.
